# SearchIt

## Description

*SearchIt then read it!*

This app is designed to allow users to search for any book that they are looking for! The results will dsplay a banner, title, along with a description. This app utilizes the MERN stack.


## Learning Points

Having been my frist big project using GraphQL, I had much to learn about utilizing mutations, queries, and providers.
